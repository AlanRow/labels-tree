# Тестовое задание: иерархическая таблица на Vue Ag Grid

Реализация тестового задания с оптимизацией TreeStore

## Установка зависимостей

```sh
npm install
```

### Запуск песочницы

```sh
npm run dev
```

### Сборка

```sh
npm run build
```

### Юнит-тесты

```sh
npm run test:unit
```

### Покрытие тестами (вью-компоненты не стал покрывать, т. к. там будет просто тестирование имеющегося библиотечного компонента)

```sh
npm run test:unit
```
